Quick start with RED - Robot Editor

Switch to Robot perspective

In order to work with RED, Robot perspective needs to be activated. This can be done in several ways, the easiest is to use top menu:
Window -> Perspective -> Open perspective -> Other and select Robot

Create Robot Project

All projects with Robot Framework files should be kept in project-type (nature) Robot, as this allows RED to activate RED specific features on Robot files. Select from top menu bar:
File -> New -> Other -> Robot Framework -> Robot Project

Empty Robot Project should looks similar as below:

If project doesn't have brown icon or no red.xml file it means that the Project is different type than Robot.
Fix this by right-click on Project name,from Robot Framework select Add Robot nature

Import Robot files into project

Import Robot files by drag&drop to created project, alternatively you can import folders by right click on project and selecting
Import -> General -> File System

Validate project and imported files

When Robot files has been imported, it is time to validate all the files to get indication about possible issues - either errors in test cases or missing libraries links in red.xml. Use Project -> Clean from top menu.
